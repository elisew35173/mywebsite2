Running Python Programs

You may have noticed that clocking on python projects doesn't take you to a site like scratch, but rather a GitHub Gist Page. To use the program, merely download the zip file. The zip file will be within a folder which will contain the python file. Then click on the python file. To replay a game, reclick the file.
