Running Python Programs

You may have noticed that clocking on pythin projects doesn;t take you to a site like scratch, but rather a GitHub Page. This is because a pythong script has to be run from the command line of your computer. You first have to download the program from GitHuband unzip the file. You can do this by downloading a programfrom the internet. make sure you save the unziped file (it should end in .py) to the desktop. The proceed to the command line. The command line can be accessed from Windows by pressing the the windows key and R. When a pop up appears, type in cmd and press enter. A black window with white print should pop up. Type cd Desktop and hit enter. Type in the name of the program in the command prompt (be sure to include the .py extension) and hit enter. The program will then begin.
